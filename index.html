Portecle Pack

This is a project that repackages the open source project Portecle to provide simple one-file executable for Windows users and .tar.gz with launcher script for Linux users.

Download:

Windows: portecle-1.7.exe
Linux: portecle-1.7.tgz

Webstart

Links to the original Portecle launcher.
