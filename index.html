Portecle Pack

This project repackages the open source project Portecle to provide a simple one-file executable for Windows and an archive with launcher script for Linux users.

Download:

Windows: Portecle-Installer-1.7.exe
Linux: portecle-1.7.tgz

Webstart

Links to the original Portecle launcher.
